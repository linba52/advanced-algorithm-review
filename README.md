# 高级算法论文 Review：Tiny Pointers

本仓库用于记录论文 *Tiny Pointers* 的详细 review 写作过程。

这个项目刻意保留写作过程，而不是只提交最终成品。仓库中的文件分工如下：

- `notes/research-log.md`：记录资料收集、论文阅读和阶段性理解过程。
- `notes/related-work.md`：记录论文相关研究，包括论文之前、论文中提到的以及论文之后的相关工作。
- `notes/technical-notes.md`：记录对模型、构造、技术直觉和证明思路的逐步理解。
- `review.md`：最终 review，由前面的研究笔记逐步发展而来。

commit 历史也是本次作业的一部分。版本记录需要展示从资料收集、逐渐理解论文、形成观点、
撰写草稿到反复修改并形成最终版本的过程。

## 当前交付物

- 最终 review：`review.md`
- 研究日志：`notes/research-log.md`
- 相关研究笔记：`notes/related-work.md`
- 技术理解笔记：`notes/technical-notes.md`
