# Advanced Algorithm Review: Tiny Pointers

This repository records the process of writing a detailed review of the paper
*Tiny Pointers*.

The project is intentionally organized to preserve the writing history:

- `notes/research-log.md` records source collection and reading progress.
- `notes/related-work.md` records related literature before and after the paper.
- `notes/technical-notes.md` records the evolving understanding of the model,
  construction, intuition, and proof ideas.
- `review.md` is the final review, developed gradually from the notes.

The commit history is part of the deliverable. Commits should show the path from
source collection, paper understanding, viewpoint formation, drafting, and
revision to the final version.
